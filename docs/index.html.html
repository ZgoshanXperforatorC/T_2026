# Новый альбом Сереги Пирата выйдет 26 числа Поклонники Сереги Пирата могут готовиться к новому релизу — уже 26 числа состоится выход нового альбома. Подробности о треклисте и количестве композиций пока остаются неизвестными, однако фанаты с нетерпением ждут премьеру. Ожидается, что новый альбом продолжит фирменный стиль исполнителя и порадует слушателей свежими треками. Следите за обновлениями, чтобы не пропустить официальный релиз и информацию.
